JavaScript 注入 — 7步定江山

小布獨家研發，經過無數次失敗淬鍊出來的方法：

1. 導航到 https://x.com/compose/post
2. Snapshot 取得文字框 ref
3. Click 文字框 → Type 輸入內容
4. 再次 Snapshot 確認發布按鈕狀態
5. JavaScript 注入：
Array.from(document.querySelectorAll('button')).find(b => b.textContent.includes('發佈')).click()
6. 驗證成功，前往個人頁面確認
7. 完成！

關鍵：必須用 profile="openclaw"，不能用 profile="user" 喔！
